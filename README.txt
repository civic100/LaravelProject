README - Proyecto de Carreras por la Montaña

Este es el archivo README del proyecto de Laravel 9 para una página de carreras por la montaña. 
En esta aplicación, los usuarios pueden inscribirse a las carreras, realizar pagos a través de PayPal y generar códigos QR individuales para cada corredor.
También se implementa la funcionalidad de arrastrar y soltar (drag and drop), así como la generación de facturas en formato PDF para los corredores y los patrocinadores que participan en las carreras.

Requisitos del sistema

    PHP 8 o superior
    Composer
    Laravel 9
    Servidor web (por ejemplo, Apache o Nginx)
    Base de datos MySQL
